### O que é o Python para Todos?

Uma **plataforma web e open-source** para ensinar pessoas iniciando no universo da tecnologia, com:

- Editor de código integrado **(CodeMirror)**
- Correção de exercícios
- Progresso salvo no navegador
- totalmente responsivo **(Funciona no celular!)**

### Como Rodar Localmente

```bash
# 1. Clone o repositório
git clone https://github.com/seu-usuario/python-para-todos.git
cd python-para-todos

# 2. Crie um ambiente virtual
python -m venv venv
source venv/bin/activate    # Linux/Mac
# ou
venv\venv\Scripts\activate     # Windows

# 3. Instale as dependências
pip install -r requirements.txt

# 4. Rode o projeto
python run.py
```

### Estrutura do projeto
```bash
python-para-todos/
├── app/
  ├── routes.py           # Todas as rotas
  ├── exercises.py        # Conteúdo dos módulos e exercícios
  ├── utils/tester.py     # Executor seguro de código
  └── templates/          # HTML com Bootstrap
├── static/               # CSS, JS e imagens
├── run.py                # Entrada da aplicação
└── requirements.txt
```
